## 📖 About The Project

**Python Assignment 2** is a curated collection of Python scripts and exercises focused on three of the most fundamental pillars of Python programming: **Functions**, **Lists**, and **Dictionaries**. 

This repository serves as a practical study guide and code reference for understanding how to manipulate data structures and write modular, reusable code. Whether you are reviewing Python basics or looking for syntax examples of specific built-in methods, this repository provides clear, functional examples.

## 🎯 Learning Objectives

By exploring the code in this repository, you will understand how to:
* **Modularize Code:** Write reusable blocks of code using functions, handle different types of arguments, and manage variable scopes.
* **Manipulate Sequences:** Traverse, slice, sort, and modify Python Lists efficiently.
* **Map Data:** Store, retrieve, and manipulate data via key-value pairs using Python Dictionaries.
